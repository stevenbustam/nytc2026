Part 01

GAMEPLAY

▸ AIM: Use of colour/code recognition and algorithm for positioning

Tasks

  • ARobot recognises red colour
  • BRobot recognises green colour and moves off start point
  • CRobot recognises April tag and centralises
  • DRobot picks up token using clamp
● STOP ● GO

Rules

  • AAll movements and decisions must be autonomous
  • BTeams may use mobile phones or lights to illuminate the April tag for better clarity
  • CAn attempt ends when the token is dropped
  • DNo other QR codes or reference signs can be added
◈ Route Diagram
Autonomous Only — Robot must detect colour signals independently. April tag centralisation required before pickup.
Part 02

GAMEPLAY

▸ AIM: Use of text recognition and line tracing

Tasks

  • ARobot recognises text "LEFT" or "RIGHT" (judge will assign)
  • BRobot moves to path corresponding to text recognised
  • CRobot navigates path using line tracing
Note: Direction assigned by judge at runtime.

Rules

  • AAll movements and decisions must be autonomous
  • BTeams may use mobile phones or lights to illuminate the text sheets for better clarity
  • CAn attempt ends when the token is dropped
  • DNo other QR codes or reference signs can be added
◈ Route Diagram
Line Tracing Required — Robot must follow the physical track on the floor. Both paths lead to Part 3.
Part 03

GAMEPLAY

▸ AIM: Use of pose recognition, face recognition and algorithm for positioning

Tasks

  • ARobot recognizes pose and moves accordingly
  • BRobot stops within Box A
  • CRobot rotates/moves autonomously and looks for correct face (judge will assign)
  • DRobot moves to correct face and centralises
  • ERobot moves to Box B and places token within Box B

Rules

  • AMovement up to Box A must be using pose recognition
  • BAll movements after reaching Box A must be autonomous
  • CTeams may use mobile phones or lights to illuminate the faces for better clarity
  • DAn attempt ends when the token is dropped
  • ENo other QR codes or reference signs can be added
◈ Route Diagram
Two-Phase Movement: Pose recognition up to Box A → Fully autonomous face detection and Box B placement after.
